找 2^k >= x 的最小 k，这可以用二分解决，但是用`__lg`更简洁，但是要把问题等价转换一下，换成 2^k > x + 1。

`__lg(x)`，x 是整型类型（integral）就行（有/无符号int / long long），但是必须 x > 0，否则 ub。

`__lg(x)`返回 x 的最高位的 1 的幂次（从 0 开始），例如 `__lg(0b100)` 返回 2。

如果是找 2^k >= x 的最小 k，则 __lg(x) 不一定是符合的，因为 x 可能还有低位 1。但是对于等价的命题 2^k > x + 1，`__lg(x + 1) + 1` 就是答案（分 x + 1 只有一个 1 / 多个 1 讨论即可）。

此外，`__lg(x) + 1` 就是 x 的 bit_width。
